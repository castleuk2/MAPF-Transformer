# Upstream references

This project is an independent implementation informed by the public interfaces
and design patterns of:

- CognitiveAISystems/MAPF-GPT (MIT): model/inference wrapper organization,
  tokenized decentralized policy interface, training entry points.
- Cognitive-AI-Systems/pogema (MIT): MAPF simulation API and action ordering.

No upstream source file is copied into this project. POGEMA integration lives in
the separate `pogema-mapf-transformer` companion package.
